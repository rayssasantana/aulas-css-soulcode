Conheça as modalidades dispoíveis

Em nossa academia, oferecemos uma variedade de modalidades aquáticas projetadas para atender às
necessidades e interesses de todos os nossos alunos. Desde aulas de natação tradicionais, ideais para aprender
desde os primeiros passos na água, até a dinâmica hidroginástica, perfeita para quem busca um exercício de
baixo impacto que fortalece o corpo inteiro. Nossos programas também incluem aulas personalizadas para
crianças, focadas em desenvolver habilidades aquáticas essenciais desde cedo, e treinamento individualizado
para quem deseja atingir metas específicas de condicionamento físico na água.

  • Aulas de Natação para todos os níveis
  • Hidroginástica para condicionamento físico
  • Aulas personalizadas para crianças
  • Treinamento Personalizado

Aulas de Natação

Oferecemos aulas de natação para todas as idades e níveis. Nossos instrutores experientes garantem um
aprendizado seguro e eficaz. Preço: R$ 150/mês.

Hidroginástica

A hidroginástica é ideal para quem busca condicionamento físico sem impacto. Nossos programas são adaptáveis
para todas as idades. Preço: R$ 130/mês.

Aulas para Crianças

Nossas aulas para crianças são especialmente desenvolvidas para ensinar natação de forma divertida e segura.
Aceitamos crianças a partir de 3 anos. Preço: R$ 120/mês.

Treinamento Personalizado

Oferecemos sessões individuais com um treinador para atender às suas necessidades específicas.
Ideal para quem busca aprimorar técnicas ou tem objetivos específicos. Preço: R$ 200/mês.
